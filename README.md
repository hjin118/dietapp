# 다이어트 기록 앱

체중, 식사, 운동, 사진을 기록하는 다이어트 관리 PWA 애플리케이션

## 기능

- 📅 **캘린더 기록** - 날짜별로 다이어트 기록 관리
- ⚖️ **체중 추적** - 체중 변화 그래프 (최근 30일)
- 🍽️ **식사 기록** - 아침, 점심, 저녁 등 식사 내용 기록
- 🏃 **운동 기록** - 운동 종류, 시간, 강도 기록
- 📸 **사진 기록** - 식단/체중 변화 사진 저장
- 🌙 **다크 모드** - 눈의 피로를 줄이는 다크 테마 지원
- 📊 **데이터 내보내기** - JSON, Markdown 형식으로 백업
- 📥 **데이터 가져오기** - 백업 파일로부터 데이터 복구
- 📱 **PWA 지원** - 오프라인 사용 및 앱 설치 가능

## 사용 방법

1. 캘린더에서 날짜를 선택합니다
2. 모달이 열리면 체중, 식사, 운동, 사진을 입력합니다
3. 저장 버튼을 클릭하여 기록을 저장합니다
4. 체중 그래프 버튼으로 최근 30일의 체중 변화를 확인합니다
5. 설정 버튼으로 데이터 내보내기/가져오기/삭제를 수행합니다

## 설치

### 로컬에서 실행
```bash
# 이 리포지토리 클론
git clone https://github.com/hjin118/dietapp.git

# 폴더로 이동
cd dietapp

# 로컬 서버 실행 (예: Live Server, http-server)
```

### PWA로 설치
- Chrome/Edge 브라우저에서 앱을 열고 주소창의 설치 아이콘 클릭

- HTML5, CSS3, JavaScript (Vanilla)
- PWA (Service Worker, Web App Manifest)
- Local Storage (데이터 저장)
- Canvas API (체중 그래프)
